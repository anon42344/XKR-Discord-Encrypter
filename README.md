# XKR-Discord-Encrypter

Go to chrome://extensions, select "Load unpacked" and select the directory with files. Navigate in your browser to a discord channel. 
Enter the password for your server, and add optional password for your channel. After you typed your message, press ยง (usually under esc button) to encrypt a message.
