# XKR-Discord-Encrypter

Go to chrome://extensions, turn on "Developer mode" select "Load unpacked" and select the directory with files. Navigate in your browser to a discord channel. Press the icon to activate. After you typed your message, press ยง (usually under esc button) to encrypt a message. 
For more security: Enter a password for your server, and add an optional password for your channel. 

<h3> Vapor mode </h3>
Disable vapor mode by right clicking icon > options > uncheck

<h3> Updates </h3>
